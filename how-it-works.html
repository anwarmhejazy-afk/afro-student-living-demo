How Afro Student Living works

Student journey

From search to confirmation

This process is built to reduce confusion, improve trust, and make student accommodation search easier across Nigeria.

🔍
Step 01

Search accommodation

Students browse by state, university area, property type, and yearly budget in Nigerian Naira.

🏠
Step 02

Compare verified listings

Students compare prices, photos, location, features, and verified badges before choosing a property.

🖼️
Step 03

Open full property details

Each listing has a detailed page with gallery images, location information, features, and booking notes.

📩
Step 04

Request booking safely

Students submit their booking request first instead of making instant payment blindly.

🛡️
Step 05

Admin reviews availability

Afro Student Living admin checks the request, property status, and next steps with the student.

✅
Step 06

Receive confirmation guidance

The student receives clear confirmation guidance before moving forward with payment or move-in plans.

For property owners

How property submission works

Property owners can submit accommodation details, but all listings remain admin-controlled.

1. Submit property details

Owners send property name, state, area, price, photos, description, and contact details.

2. Admin verifies

The admin checks whether the property is suitable, clear, and safe for student listing.

3. Approved listings go live

Only approved properties appear on the public accommodation listings page.
