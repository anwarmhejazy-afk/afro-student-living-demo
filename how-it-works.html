How Afro Student Living works

A safer way to find and request student accommodation in Nigeria.

Afro Student Living is built to help students compare accommodation, request bookings, and avoid unsafe or unverified property arrangements.

01

Search by state, area, or university

Students can browse accommodation by Nigerian state, city area, university location, property type, and yearly budget in Nigerian Naira.

  • Search by state such as Lagos, Enugu, Rivers, Kano, or Abuja
  • Search by university area such as UNILAG, UNEC, or University of Abuja
  • Filter by studio, hostel, apartment, or shared room
  • Compare yearly prices in ₦ Nigerian Naira
02

Compare verified accommodation options

Each listing is presented with key student-focused details so students can compare before sending a request.

  • Property images
  • Location and nearby university
  • Price per year
  • Property features such as WiFi, security, water, power, and furnishing
  • Verified or featured listing badges
03

Open full property details

Students can open a property page to see more details before making any booking decision.

  • Large gallery-style property images
  • Full property description
  • Accommodation features
  • Booking note and admin confirmation information
  • WhatsApp contact option
04

Request booking safely

Instead of paying immediately, students submit a booking request first. This helps reduce risk and gives the admin time to confirm availability.

  • Student name and contact details
  • University information
  • Preferred move-in date
  • Optional message or questions
  • Booking status tracking through student account
05

Admin reviews the request

Afro Student Living admin checks the booking request, verifies the property status, and confirms next steps with the student.

  • Admin reviews student booking requests
  • Admin checks property availability
  • Admin follows up with student or property contact
  • Admin updates booking status
06

Student receives confirmation guidance

Once availability is confirmed, the student receives guidance on the next step before moving forward.

  • Confirmation of availability
  • Clear communication before payment
  • Move-in guidance
  • Support through WhatsApp or admin contact
For property owners

How property submission works

Property owners can submit accommodation details, but they do not get public admin access. Afro Student Living admin reviews everything before publishing.

1. Submit property details

Owners send property name, state, area, price, photos, description, and contact details.

2. Admin verifies

The admin checks whether the property is suitable, clear, and safe for student listing.

3. Approved listings go live

Only approved properties appear on the public accommodation listings page.
